## Visão geral

### Captura de tela

![](./screenshot.jpg)

Adicione uma captura de tela da sua solução. A maneira mais simples é usar o Firefox para visualizar o projeto, clicar com o botão direito e escolher "Capturar tela". Você pode escolher captura completa ou recortada, dependendo do tamanho da página. Se for muito longa, pode ser melhor recortar.

Outra opção é usar uma ferramenta como [FireShot](https://getfireshot.com/) para capturar a tela. O FireShot tem opção gratuita, então você não precisa comprar.

Depois, recorte/otimize/edite a imagem como quiser, adicione ao projeto e atualize o caminho na imagem acima.
